控制台中访问节点 ​

  • $_ 返回最近一次表达式的结果
  • $0 表示当前选中的元素
  • $1, $2, $3...表示之前选中的元素。
  • $ 查询选择器
  • $$ 查询选择器的所有元素
  • $x xpath 定位元素

列出分配给元素的事件侦听器 ​

getEventListeners() 查看元素绑定的事件

javascript
getEventListeners($0);

监视元素上的事件 ​

monitorEvents

javascript
monitorEvents(document.querySelector("img"));
monitorEvents(document.querySelector("img"), ["mousedown", "mouseup", "click"]);
monitorEvents(window, ["resize", "scroll"]);
unmonitorEvents(window, ["resize", "scroll"]);

console.dir(obj) 检查对象

console.trace() 输出调用栈

console 常用的方法 ​

  • log()
  • info()
  • warn()
  • error()
  • count()
  • table()
  • clear()
  • assert() 断言输出
  • time() timeEnd() 计时器开始和结束
  • profile() profileEnd() 性能分析开始和结束

实时表达式监视 ​

javascript
let x = 0;
let y = 0;
document.addEventListener("mousemove", (e) => {
  x = e.x;
  y = e.y;
});
